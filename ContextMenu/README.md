# Mục đích
Tạo context menu customize

# File
background.reg --> File add customize context menu khi click chuột phải
```
TuPT Context
    - Git Gui Here
    - Git Bash Here
    - Open Visual Code
```

shell.reg --> File add customize context menu khi click chuột phải vào folder
```
TuPT Context
    - Git Gui Here
    - Git Bash Here
    - Open Visual Code
```
